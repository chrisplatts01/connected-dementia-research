Connect Dementia Research
Pattern Library

Component and module library for the Connect Dementia Research project website.

Reusable components and modules, linked in the menu on the left, are illustrated with a description, one or more examples, the HTML code used for the example and the SCSS code used to generate the CSS styling for that example.

Where dynamic styling needs to be illustrated, for instance hover and checked states, they are simulated using classes. These classes are not required in production code.
